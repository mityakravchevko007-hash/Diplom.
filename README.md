1. Обзор проекта
Данный проект представляет собой веб-приложение интернет-магазина, разработанное на Django. Основной функционал включает каталог товаров, корзину, избранное и систему пользователей.
2. Функциональность
- Каталог товаров (products)
- Добавление в корзину (cart)
- Избранное (favorites)
- Регистрация и авторизация пользователей (accounts)
- Отзывы (reviews)
3. Структура проекта
Основные приложения:
- accounts – пользователи
- products – товары
- cart – корзина
- favorites – избранное
- reviews – отзывы

4. Интерфейс
В проекте используются HTML-шаблоны:
- base.html – основной шаблон
- cart.html – корзина
- favorites.html – избранное
- login.html – авторизация
